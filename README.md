# Jens's fish files

![](https://raw.githubusercontent.com/rstacruz/fishfiles/gh-pages/screenshot.png)

Clone:

```bash
mkdir -p ~/.config
git clone https://github.com/jpfeifle/fishfiles.git ~/.config/fish
```

In a new shell:

```bash
# Install dependencies
fisher

# Load abbreviations
source ~/.config/fish/abbreviations/all.fish
```
